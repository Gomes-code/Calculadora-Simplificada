# Notas de identidade visual aplicadas

A interface usa como base a paleta e diretrizes do manual de marca TPF:

- Azul TPF: `#0067FF`
- Azul escuro: `#00284D`
- Azul de apoio: `#0096FF`, `#5ABDFE`, `#BCE4FE`
- Cinzas: `#1D1D1B`, `#636362`, `#E6E7E8`
- Laranja de apoio: `#FD6B3F`
- Tipografia: Aptos com fallback Roboto/Arial

A marca original não foi reconstruída como arquivo de imagem. O cabeçalho usa um elemento geométrico inspirado na rosácea apenas como recurso visual de interface.
